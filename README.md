# Sega/Gremlin Zaxxon port for MiSTer

# Keyboard inputs :
```
   ESC         : Coin 1
   F1          : Start 1P
   F2          : Start 2P 
   UP,DOWN,LEFT,RIGHT arrows : Player 1
   LCtrl : Fire A


   MAME/IPAC/JPAC Style Keyboard inputs:
     5           : Coin 1
     6           : Coin 2
     1           : Start 1 Player
     2           : Start 2 Player
     R,F,D,G     : Player 2
     A           : Fire2 A
	
 Joystick support. 
```
